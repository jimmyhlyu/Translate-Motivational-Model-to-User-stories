Model: I got you!

Background:
The project aims to handle intruders and protect the owner's property by detecting intruders, notifying stakeholders, and responding to intrusions. The emotional goals include accurate, safe, and timely actions to achieve the project's objective.

Epic 1: Detect Intruder
Security Manager:
- As a security manager, I can configure motion sensors to detect intruders accurately, so that I can receive real-time alerts.

Visitor:
- As a visitor, I can scan my unique QR code to identify myself and avoid triggering false alarms, so that I can gain access to the property safely.

Epic 2: Notify Stakeholder
Security Manager:
- As a security manager, I can receive immediate intrusion notifications on my mobile device, so that I can take immediate action to protect the property.

Police:
- As the police officer, I can view real-time security footage of the intrusion on my device, so that I can respond promptly and appropriately.

Epic 3: Response to Intrusion
Security Manager:
- As a security manager, I can remotely activate the barrier and siren to deter intruders, so that I can protect the property effectively.

Place Owner:
- As the place owner, I can grant remote access to the security manager to take actions when necessary, so that I can ensure the property's safety 24/7.
