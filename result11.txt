
 Model: 1 
Model: I got you!
Background: The goal is to handle intruders and protect owner's property.

Epic 1: Detect Intruder 
Security manager: 
- As a security manager, I can receive real-time alerts if any sensors in the area indicate an intrusion, to feel safe about the monitoring of the area. 
- As a security manager, I can view a live video stream of the triggered sensor, so that I can confirm whether there is an actual intrusion or not with accuracy.

Visitor:
- As a visitor, I can scan a QR code that is displayed at the entrance to notify the security manager of my presence to avoid the intrusion detection system, to ensure I will not trigger any false alarms.

Epic 2: Notify Stakeholder 
Police: 
- As police officers, we can receive notifications from the security system when there is an intrusion, and automatically dispatch a unit to the location immediately, so that we can respond to the situation as soon as possible. 

Stakeholder 2:
- As a stakeholder 2, I can receive an SMS alert from the security system if there is any potential intrusion, so that I can respond the intrusion as soon as possible.

Epic 3: Response to intrusion 
Place owner: 
- As the place owner, I can remotely enable/disable the alarm system after confirming the intrusion, to protect my property. 
- As the place owner, I can track any past intrusion events along with footage recorded in real-time to maintain the security of the premises.

Intruder: 
- As an intruder, I can hear audible alarms, and see flashing lighting to alert me that I have triggered the intrusion detection system, so I can avoid or flee the premises.

Emotional goals: 
Accurate - Ensure the intrusion detection system is accurate and reliable
Safe - Stakeholders feel safe and protected knowing the security system is in place
Timely- Security system can respond in real-time to an intrusion.

Epic 5: Publishing exercises and workout plans
Gym-goers:
- As a gym-goer, I can share my exercise and workout plans with my friends or other users, so that I can receive feedback and make my plan more effective.

- As an administrator, I can analyze and moderate user-submitted exercises and workout plans, so that there are no potentially harmful or indecent user-generated content available to others.
